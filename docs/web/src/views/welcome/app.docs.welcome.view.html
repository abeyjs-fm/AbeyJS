Why AbeyJS?

  • Declarative

    Less ceremony: stable configuration and OM views instead of scattered layers with no contract.

  • Generated UI

    Lists and forms aligned to data and APIs; compiler and CLI scaffold the boring parts.

  • Omega runtime

    A single intent channel between view, shell, and services—without hand-coupling DOM details.

  • Modular

    Focused @abeyjs/* packages—bring in only what your app needs.

  • XSS-conscious

    An explicit contract for sanitized HTML where it matters—documented in the security guides.
