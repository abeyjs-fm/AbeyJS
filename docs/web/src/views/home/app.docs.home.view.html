AbeyJs documentation

Documentation from the same stack you run in production

Declarative routes, compiled OM views (*.view.html), AbeyJS runtime (OmegaRuntime), and intents over the channel. Guides written from the monorepo, with clear limits and direct links to the code where it matters.

This site is AbeyJs documentation only. Dark chrome / violet–blue branding and wide layout are CSS for this site (docs/web/src/styles). The shell from @abeyjs/view serves routes, outlet, and menu; you theme it with variables and stylesheets (e.g. themeVarsLight/Dark, your own theme): other apps do not inherit the same look by default.
